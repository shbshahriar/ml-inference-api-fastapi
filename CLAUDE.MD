# CLAUDE.md

This file provides guidance to Claude Code when working with this repository.

## Project Overview

Structured learning repository for building REST APIs with FastAPI ‚Äî from basic routing to deploying ML models in production. Organized into 11 numbered topic folders, each with Python scripts, markdown guides, and interactive HTML references.

GitHub: `https://github.com/shbshahriar/ml-inference-api-fastapi`

## Environment & Commands

This project uses **uv** exclusively ‚Äî never use `pip`.

```bash
uv add <package>                                          # add dependency
uv run uvicorn 04_path_query_params.path_query_api:app --reload  # run a lesson
```

- Python: **3.13** (pinned in `.python-version`)
- Core deps: `fastapi>=0.135.3`, `uvicorn>=0.43.0`, `pydantic>=2.12.5`
- uv manages `.venv/` automatically

### Uvicorn module path convention

```bash
uv run uvicorn <folder>.<file>:app --reload
```

## Data

- Shared dataset lives at `Data/patients.json` ‚Äî a dict keyed by string IDs (`"p001"`, `"p002"`, ...)
- Always use `pathlib.Path(__file__)` to build file paths ‚Äî never plain relative strings
- `pathlib.Path` conflicts with `fastapi.Path` ‚Äî alias one: `from pathlib import Path as filePath`

## Folder Structure

| Folder | Topic |
|--------|-------|
| `01_what_is_api/` | What is an API ‚Äî interactive HTML guides |
| `02_fastapi_setup/` | First app, server commands |
| `03_http_methods/` | GET, POST, PUT, DELETE |

| `06_put_delete_requests/` | PUT and DELETE |

| `09_docker_for_ml/` | Dockerfile, `docker build`, `docker run` |

`main.py` at root is a stub ‚Äî working files live inside topic folders.

## Constraints

- **Pydantic**: do not explain unless explicitly requested
- **ML projects**: only Iris Classification, House Price Prediction, Insurance Cost Prediction, Customer Churn Prediction
- **Model serialization**: only `pickle` or `joblib`
- **Docker**: only Dockerfile, `docker build`, `docker run` ‚Äî no orchestration
- **Deployment targets**: Render, Railway, Heroku, AWS, GCP, Azure
- Do not introduce frameworks or topics outside this scope

- Keyboard navigation (‚Üê ‚Üí arrow keys)
